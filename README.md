# Spotify Clone

Một bản dựng giao diện và chức năng cơ bản của ứng dụng Spotify, được tạo để học tập và thực hành Frontend / Web Development.

## 📌 Mô tả dự án

Dự án này là một bản **Spotify Clone** mô phỏng giao diện và trải nghiệm nghe nhạc tương tự Spotify, được xây dựng bằng **HTML, CSS và JavaScript** (hoặc framework bạn đang dùng). Mục tiêu là tạo ra một giao diện trực quan, dễ dùng và thể hiện được các tính năng chính như:

- Trang đăng nhập
- Trang danh sách bài hát / playlist
- Thanh điều khiển phát nhạc
- UI responsive cho nhiều kích thước màn hình :contentReference[oaicite:0]{index=0}

> 🔄 **Lưu ý:** Nếu dự án của bạn sử dụng API (ví dụ Spotify API), bạn có thể thêm phần hướng dẫn lấy key và cách cấu hình vào phần cài đặt bên dưới.

## 🛠️ Công nghệ sử dụng

Danh sách thư viện, framework và công cụ trong dự án này:

- HTML5 – cấu trúc trang
- CSS3 – tạo kiểu và layout
- JavaScript (hoặc framework như React / Vue) – xử lý logic & tương tác
- (Nếu dùng) Tailwind CSS – utility CSS
- (Nếu dùng) Spotify Web API – lấy dữ liệu nhạc thật :contentReference[oaicite:1]{index=1}
